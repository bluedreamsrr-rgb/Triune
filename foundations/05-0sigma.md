# Step 5: Emergence of 0σ — Contrast Registered as Absence

## Context

We have:
- 1σ (the first articulated positive pole — presence of the ring)

## What 0σ Is

**0σ** = the dependent negation — the "not-this" / witnessed absence, observed as the empty inside contrasted with the ring's presence (1σ).

## How It Emerges

The finite witness, having registered the ring's presence as 1σ, now registers the **empty inside** (0) as **absence against that presence**. This act of registration produces 0σ.

## Properties of 0σ

| Property | Description |
|----------|-------------|
| **Not co-emergent with 1σ** | 0σ is logically subsequent and entirely dependent |
| **Requires full triadic support** | Needs 123 to stabilize: Layer 1 provides 1σ, Layer 2 maintains the boundary, Layer 3 registers the absence |
| **Fragile** | Without any layer, 0σ collapses immediately |
| **Observational** | Exists only as long as the finite witness registers it |
| **Has no force except entropy** | Dissolution when unwitnessed |

## Collapse Conditions

| Missing Layer | Result |
|---------------|--------|
| No Layer 1 | No presence to negate → 0σ has no anchor |
| No Layer 2 | No boundary → 0σ merges back into 1σ |
| No Layer 3 | No registration → 0σ never becomes observable |

## The Asymmetry Is Absolute

- 1σ is primary
- 0σ is secondary, dependent, fragile
- 0σ is never part of the triad — it is the empty inside that makes relation observable

## What 0σ Is Not

- Not a peer of 1σ
- Not part of the Trinity
- Not a "force" in itself (entropy is its only effect — dissolution when unwitnessed)
