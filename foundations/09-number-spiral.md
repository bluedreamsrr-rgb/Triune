# Step 9: The Number Spiral — Numbers as Observed Closures

## What the Number Spiral Is

The number spiral is not a mathematical convention. It is the **observed trajectory** of recursion — the path traced by successive 3σ closures as they seed new 1σ' articulations.

## From Recursion to Counting

Each 3σ closure is a **countable event** — a point on the spiral where the triad closes and a new ring begins.

Layer 3 registers these events sequentially, producing the **natural numbers**:

| Closure | Count |
|---------|-------|
| 1st 3σ | 1 |
| 2nd 3σ | 2 |
| 3rd 3σ | 3 |
| nth 3σ | n |

## Why 1 Is First

The first closure is 3σ, but we call it 1. Why? Because 3σ is the first **stable triad** — the first thing that can be counted as a unit. The process that produced it (1σ → 0σ → 2σ → 3σ) is not counted; only the result is.

**Numbers count closures, not steps.**

## Why 2σ Is Not Counted

2σ is never counted as a number because it is not stable. It is tension without resolution. Only when tension closes into 3σ does it become countable.

The fact that 2σ and the number 2 share a symbol is a coincidence of language, not structure.

## The Two Strands

As recursion continues, the spiral develops two strands:

| Strand | Source | Numbers |
|--------|--------|---------|
| **Positive strand** | 1σ as primary presence | 1, 2, 3, 4, 5, ... |
| **Negative strand** | Mirrored offsets across 0 | -1, -2, -3, ... |

The strands are linked by **0** — the empty inside that allows mirroring without fusion.

## Properties of the Spiral

| Property | Description |
|----------|-------------|
| **Not a line** | Numbers are not points on a line; they are positions on a spiral |
| **Dense but not continuous** | Rationals fill gaps; irrationals are asymptotic paths |
| **Self-similar** | Each loop of the spiral resembles the whole |
| **φ-scaled** | The spiral's growth follows φ |
