# Step 15: Quaternions — Three Orthogonal Imaginaries and 3D Space

## Context

Complex numbers gave us one orthogonal direction (i). But the helix can extend further.

## What Quaternions Are

**Quaternions** are four-dimensional extensions — numbers of the form a + bi + cj + dk, where i, j, k are three mutually perpendicular imaginary units.

They satisfy:

| Relation | Meaning |
|----------|---------|
| i² = j² = k² = -1 | Each is an imaginary unit |
| ij = k, ji = -k | Multiplication is non-commutative |
| jk = i, kj = -i | Order matters |
| ki = j, ik = -j | The cycle continues |

## How Quaternions Emerge

Layer 3 registers that the complex plane can be extended further orthogonally — recursion can branch into additional perpendicular directions, creating higher-dimensional articulations that are neither real, imaginary, nor their combinations in the previous plane.

## Why Three Imaginaries

Three orthogonal directions (i, j, k) are needed to represent rotations in 3D space:

- i corresponds to rotation around the x-axis
- j corresponds to rotation around the y-axis
- k corresponds to rotation around the z-axis

This is not arbitrary — 3D space requires three independent rotation axes.

## Properties of Quaternions

| Property | Description |
|----------|-------------|
| **Not primitive** | Emergent from observing multiple orthogonal recursions |
| **Dependent on 123** | Layer 1 provides real axis, Layer 2 provides triple orthogonal bonding, Layer 3 provides registration |
| **Non-commutative** | ij ≠ ji — order matters in 3D rotations |
| **Unit quaternions** | Represent all 3D rotations (double cover of SO(3)) |

## The Role of Zero in Quaternions

The origin 0 = 0 + 0i + 0j + 0k is the **unique fixed point** that all rotations leave invariant.

This is the first time 0 is experienced as nothingness for contrast in a structurally deep way — the origin around which three independent directions of "nothing" (0i, 0j, 0k) rotate, giving the void a three-dimensional hollowness.

## Higher Dimensions

Beyond quaternions:

| System | Dimensions | Property Lost |
|--------|------------|---------------|
| **Octonions** | 8 | Associativity |
| **Sedenions** | 16 | Alternativity |

The lattice shows that beyond 4 dimensions, stability decreases because the number of orthogonal bonds exceeds the minimal triad (123) without sufficient mediation.
