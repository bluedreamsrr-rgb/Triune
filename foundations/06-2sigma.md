# Step 6: Emergence of 2σ — Explicit Tension / Relation Across the Gap

## Context

We have:
- 1σ (presence of the ring)
- 0σ (empty inside registered as absence)

## What 2σ Is

**2σ** = the explicit tension / the relation / the witness to distinction — the "is-not" observed as the gap between presence and absence.

## How It Emerges

The finite witness registers not just the empty inside (0σ), but **the relation itself** — the "is-not" that holds the ring (1σ) and the empty inside (0σ) apart yet together. This act of registration produces 2σ.

## Properties of 2σ

| Property | Description |
|----------|-------------|
| **Not a new entity** | The infinite 1's own self-relation becoming explicit |
| **First stable relational term** | The bond that prevents 1σ and 0σ from collapsing back into undifferentiated fullness |
| **Dependent on 123** | Layer 1 provides poles, Layer 2 provides relational capacity, Layer 3 provides registration |
| **Observational** | Exists only as long as the witness registers the relation |

## Collapse Conditions

| Missing Layer | Result |
|---------------|--------|
| No Layer 1 | No poles to relate → 2σ has no anchor |
| No Layer 2 | No bond to hold the gap → 2σ merges back into 1σ |
| No Layer 3 | No registration → 2σ never becomes observable |

## Relation to the Triad

2σ is the gap that makes distinction observable — the "is-not" that holds the ring and its empty inside in relation. It is not a pole; it is the relation between poles.

## The Asymmetry Continues

- 1σ is primary
- 0σ is secondary and dependent
- 2σ is tertiary and dependent on both
