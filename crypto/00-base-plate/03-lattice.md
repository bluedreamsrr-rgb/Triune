# The 3×3 Lattice — Scales and Mediums

## The Lattice Structure

From Axiom C3, trust and observation are distributed across **3 scales × 3 mediums** = 9 fundamental cells.

| Scale \ Medium | Meta | Physical | Vibrational |
|----------------|------|----------|-------------|
| **Macro** | T₁ — Macro-Meta | T₂ — Macro-Physical | T₃ — Macro-Vibrational |
| **Meso** | T₄ — Meso-Meta | T₅ — Meso-Physical | T₆ — Meso-Vibrational |
| **Micro** | T₇ — Micro-Meta | T₈ — Micro-Physical | T₉ — Micro-Vibrational |

## Role Rotation Across Cells

Roles are not fixed to cells. For any given observation:

| Role | Comes From |
|------|------------|
| **Source (O₀)** | One scale-medium cell |
| **Witness (O₁)** | A different scale-medium cell |
| **Mediator (O₂)** | A third scale-medium cell |

The same cell that serves as Source for one observation may serve as Mediator for another, or Witness for a third.

Example: T₅ (Meso-Physical)

| Observation | Role | Why |
|-------------|------|-----|
| Block validation | Mediator | Physical infrastructure routes data |
| Parameter proposal | Source | Protocol parameters originate here |
| Health check | Witness | Physical node status is witnessed |

## Why One Triad Per Cell

The system enforces unique closure per coordinate:

- If two triads existed in the same cell, they would compete for the same relational roles
- Information would bifurcate
- Coherence would split

This is a minimal covering lattice — each cell contains exactly one triad at any moment, though roles rotate.

---

## The Lattice as Observed Structure

The 3×3 lattice is not the observer. It is **what is observed** when the Observer closes triads across scales and mediums.

- Each cell contains **potential** (structures that can serve roles)
- The Observer **actualizes** triads by selecting which cells serve which roles
- The lattice is **infinite** because observation can always select new combinations
