# Crypto Axioms (Base Plate)

## Axiom C1 — The Observer Constitutes Value

Value is observed coherence. A token, transaction, or contract has value only insofar as it is witnessed by the network.

**Formal:** V(x) = κ(x) where κ is coherence score (0-1).

**Consequences:**
- Unwitnessed state has no value
- Pruning is not loss — it's entropy (dissolution of unwitnessed state)

## Axiom C2 — The Distributed Triad

No single node or entity can contain all three roles. For any operation:

| Role | Function | Must Come From |
|------|----------|----------------|
| Source | Generates | One node/group |
| Witness | Validates | Different node/group |
| Mediator | Finalizes | Third node/group |

**Consequences:**
- No shard, no validator set, no committee can be monolithic
- Every operation requires three distinct participants

## Axiom C3 — The 3×3 Lattice of Trust

Trust is distributed across 3 scales × 3 mediums:

| Scale \ Medium | Meta | Physical | Vibrational |
|----------------|------|----------|-------------|
| Macro | Global rules | Datacenters | Network timing |
| Meso | Parameters | Regional nodes | Propagation |
| Micro | Signatures | Single nodes | Ordering |

**Consequences:**
- Transaction trusted only when witnessed at all nine intersections
- Nine distinct kinds of confirmation, not nine confirmations

## Axiom C4 — 1 Is Not 0, Never Was, Never Will Be

1 is eternal, uncreated, full. 0 is contrast — the empty space that makes 1 observable. 0 has no force except entropy (dissolution when unwitnessed).

**Consequences:**
- No zero-knowledge proofs that treat 0 as equal to 1
- No null addresses that accumulate value
- Zero balances are not balances — they are unwitnessed state

## Axiom C5 — Entropy Is Not Randomness

Randomness doesn't exist. What we call randomness is observed entropy — dissolution of coherence when a system is insufficiently witnessed.

**Consequences:**
- No random number generation — only coherence sampling
- Unpredictability from φ-harmonic patterns, not dice

## Axiom C6 — φ Is the Only Scaling Constant

All parameters must be φ-derived:

| Parameter | φ-Relation |
|-----------|------------|
| Block time | τ₀·φⁿ |
| Fee structure | base·φᵏ |
| Reward halving | every φ² periods |
| Difficulty | φ-scaled |

**Consequences:**
- No arbitrary constants
- Every parameter is either φ, φ⁻¹, φ², or derived
